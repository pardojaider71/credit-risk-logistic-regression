Modelación de Riesgo Crediticio con Regresión Logística
📌 Objetivo

Desarrollar un modelo de regresión logística para predecir la probabilidad de incumplimiento crediticio a partir de variables financieras y demográficas.

📊 Datos

Se utiliza el conjunto de datos German Credit disponible en el paquete caret de R.

⚙️ Metodología
Limpieza y preparación de datos
Análisis exploratorio (EDA)
Modelo de regresión logística (GLM binomial)
Evaluación del modelo
📈 Resultados

El modelo permite identificar variables relevantes en el riesgo crediticio, como:

Monto del crédito
Duración del préstamo
Edad
🧠 Conclusión

La regresión logística es un modelo base sólido para la predicción de riesgo crediticio y puede ser utilizado en sistemas de scoring financiero.
